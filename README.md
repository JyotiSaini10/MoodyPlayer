# 🎵 Moody Player

Moody Player is a full-stack web application that detects a user's facial expression and recommends songs based on the detected mood.

## 🚀 Features

- 😊 Real time facial emotion detection using Face API.js
- 🎵 Mood based music recommendations
- ☁️ Audio upload and storage using ImageKit
- 📂 Song management with MongoDB
- 🔗 RESTful APIs built with Express.js
- ⚡ Responsive React frontend

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- Face API.js

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Cloud Storage
- ImageKit
